### **Atelier Pratique : Connexion et Déconnexion d'un Utilisateur avec une Base de Données en PHP avec PDO**  

Cet atelier a pour objectif de mettre en pratique la gestion des **cookies**, des **sessions**, et de la **connexion sécurisée avec une base de données** en PHP en utilisant **PDO**.  

---

## **Objectifs de l'Atelier**  

Nous allons réaliser une application web simple permettant :  
1. **Inscription d'un utilisateur** avec hachage sécurisé du mot de passe.  
2. **Connexion d'un utilisateur** avec vérification du mot de passe via `password_verify()`.  
3. **Mémorisation avec un cookie** si l'utilisateur choisit "Se souvenir de moi".  
4. **Gestion des sessions** pour maintenir l'utilisateur connecté.  
5. **Déconnexion sécurisée** avec suppression des sessions et des cookies.  
6. **Redirection et sécurisation des pages protégées**.  

---

## **1. Création de la Base de Données (`gestion_utilisateurs.sql`)**  

```sql
CREATE DATABASE gestion_utilisateurs;
USE gestion_utilisateurs;

CREATE TABLE utilisateurs (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL UNIQUE,
    email VARCHAR(100) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL
);
```

### **Explication du Code :**  
- `username` est unique pour éviter les doublons.  
- `email` est également unique.  
- `password` stocke un mot de passe haché pour plus de sécurité.  

---

## **2. Connexion à la Base de Données avec PDO (`db.php`)**  

```php
<?php
function getConnection() {
    try {
        $pdo = new PDO("mysql:host=localhost;dbname=gestion_utilisateurs;charset=utf8", "root", "", [
            PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION, 
            PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC
        ]);
        return $pdo;
    } catch (PDOException $e) {
        die("Erreur de connexion : " . $e->getMessage());
    }
}
?>
```

### **Explication du Code :**  
- Connexion sécurisée avec **gestion des erreurs activée** (`ERRMODE_EXCEPTION`).  
- Les résultats sont récupérés en **tableau associatif** (`FETCH_ASSOC`).

## **3. Formulaire d'Inscription (`register.php`)**  

```php
<?php
require 'db.php';

if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $pdo = getConnection();
    $username = trim($_POST['username']);
    $email = trim($_POST['email']);
    $password = password_hash($_POST['password'], PASSWORD_DEFAULT);

    $stmt = $pdo->prepare("INSERT INTO utilisateurs (username, email, password) VALUES (?, ?, ?)");
    if ($stmt->execute([$username, $email, $password])) {
        echo "Inscription réussie ! <a href='login.php'>Connectez-vous ici</a>";
    } else {
        echo "Erreur lors de l'inscription.";
    }
}
?>

<form method="POST">
    <input type="text" name="username" required placeholder="Nom d'utilisateur">
    <input type="email" name="email" required placeholder="Email">
    <input type="password" name="password" required placeholder="Mot de passe">
    <button type="submit">S'inscrire</button>
</form>
```

### **Explication du Code :**  
- **Hachage du mot de passe** avec `password_hash()`.  
- **Insertion sécurisée avec PDO** (`prepare()` et `execute()`).

## **4. Formulaire de Connexion (`login.php`)**  

```php
<?php
session_start();
require 'db.php';

if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $pdo = getConnection();
    $username = trim($_POST['username']);
    $password = trim($_POST['password']);
    
    $stmt = $pdo->prepare("SELECT * FROM utilisateurs WHERE username = ?");
    $stmt->execute([$username]);
    $user = $stmt->fetch();

    if ($user && password_verify($password, $user['password'])) {
        $_SESSION['username'] = $user['username'];
        
        if (isset($_POST['remember'])) {
            setcookie("utilisateur", $user['username'], time() + 86400, "/");
        }

        header("Location: dashboard.php");
        exit;
    } else {
        echo "Identifiants incorrects.";
    }
}
?>

<form method="POST">
    <input type="text" name="username" required placeholder="Nom d'utilisateur">
    <input type="password" name="password" required placeholder="Mot de passe">
    <label>
        <input type="checkbox" name="remember"> Se souvenir de moi
    </label>
    <button type="submit">Se connecter</button>
</form>
```

### **Explication du Code :**  
- **Vérification du mot de passe** avec `password_verify()`.  
- **Utilisation de sessions pour stocker l'utilisateur connecté**.  
- **Mémorisation avec un cookie si l'option "Se souvenir de moi" est cochée**.

## **5. Page Protégée (`dashboard.php`)**  

```php
<?php
session_start();
if (!isset($_SESSION['username']) && !isset($_COOKIE['utilisateur'])) {
    header("Location: login.php");
    exit;
}

$username = $_SESSION['username'] ?? $_COOKIE['utilisateur'];
?>

<h1>Bienvenue, <?= htmlspecialchars($username) ?></h1>
<a href="logout.php">Déconnexion</a>
```

### **Explication du Code :**  
- Vérifie si **une session ou un cookie est défini**.  
- Si l'utilisateur est connecté, on affiche son nom.  
- Utilisation de `htmlspecialchars()` pour éviter les attaques XSS.  

---

## **6. Déconnexion et Suppression des Sessions et Cookies (`logout.php`)**  

```php
<?php
session_start();
session_destroy();
setcookie("utilisateur", "", time() - 3600, "/");
header("Location: login.php");
exit;
?>
```

### **Explication du Code :**  
- `session_destroy()` supprime toutes les données de session.  
- `setcookie("utilisateur", "", time() - 3600, "/")` efface le cookie.  
- Redirection vers la page de connexion après la déconnexion.
